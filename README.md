# The Virtual Vault: E-Commerce Analytics Project

## Project Overview
This project leverages a synthetic dataset designed for a fictional e-commerce platform, **The Virtual Vault**.  
The dataset simulates real-world online retail scenarios, capturing customer demographics, purchase behavior, product categories, and returns.

**Objective:** Demonstrate how data analytics can uncover actionable insights that drive business growth and customer satisfaction.  
The analysis explores **sales trends, customer segmentation, and operational efficiencies** to support strategic decision-making.

---

## Dataset Overview
The dataset consists of multiple tables representing different aspects of the business:

| **Table**     | **Description** |
|---------------|-----------------|
| **Orders**    | Order details (Order ID, Date, Amount, Customer ID, Product ID) |
| **Customers** | Customer demographics (Customer ID, Name, Country, City) |
| **Products**  | Product details (Product ID, Name, Category, Sub-category, Price) |
| **Returns**   | Return records (Order ID, Customer ID, Return Reason) |

---

## Dashboards Created

### 1. Sales Performance Dashboard
- Identifies key revenue drivers (Top N products) contributing most to total sales  
- Tracks total sales by year, analyzing **YoY% growth** and long-term trends  
- Maps sales footprint across cities to highlight expansion and high-revenue locations  
- Analyzes orders by hour to identify **peak sales periods** and customer purchasing behavior  

### 2. Products Dashboard
- Shows category contribution to total revenue, highlighting profitable product categories  
- Tracks total orders by category to identify high-demand segments  
- Monitors stock status to ensure inventory availability and prevent stockouts  
- Highlights Top N sub-categories by orders to optimize inventory and marketing strategies  

### 3. Customers Dashboard
- Visualizes total orders across countries to identify global sales trends  
- Breaks down orders across cities for targeted growth strategies  
- Highlights Top N countries by sales, showcasing high-revenue regions for expansion  
- Includes a customer details table (Name, Country, City, etc.) for deeper customer insights  

### 4. Returns Dashboard
- Country-wise return analysis to identify regions with high return rates and adjust policies  
- Product-wise return analysis to highlight items with frequent returns for quality control  
- Return reasons breakdown (e.g., defective items, wrong product received) to reduce return rates and improve satisfaction

## Business Impact
Through these dashboards, **The Virtual Vault** gains:
- Clear visibility into **sales performance and growth trends**  
- Insights into **product demand and inventory management**  
- Better understanding of **customer demographics and purchasing behavior**  
- Data-driven strategies to **reduce returns and improve customer satisfaction**
